【COMBATについて】
■操作
・wasd 		移動
・space 	回避
・↑←↓→　	弾の発射
・Eneter	シーン遷移

■制作期間2ヵ月半

■紹介
DirectXを使用して制作した3Dアクションゲームです。
ゲームとしては、アニメーションを付ける事によって、3Dゲームならではの表現をしました。
また、ウェーブ処理、移動制御、カメラのシェイクを組み合わせることで、スタン攻撃を実装しました。
技術面では、継承を用いたポリモーフィズム、ステートパターン、簡易的なフレームワークを実装しています。
制作時は、コメントや命名規則を意識しながら進めました。

■改善点
・シェーダーへの理解を深め、影や演出のクオリティを上げること。
・アニメーションのプログラムが解読し辛いのを改善すること。
